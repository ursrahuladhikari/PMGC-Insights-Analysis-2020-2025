# PMGC Multi-Year Analysis Dashboard (2020-2025)

A comprehensive, data-driven esports dashboard for the **PUBG Mobile Global Championship (PMGC)**. This platform provides historical analysis, team performance transitions, and tournament insights spanning from the inaugural 2020 edition up to the 2025 championship.

## 🚀 Key Features
- **Dynamic Year Navigation**: Seamlessly switch between tournament years (2020-2025) using the hero navigation symbols or the sticky navbar.
- **Regional Distribution**: Interactive charts showing team participation by region (SEA, PEL, EMEA, etc.) for each specific year.
- **Map Pool Insights**: Visual analysis of the official map rotations, including the introduction of **Rondo** in 2025.
- **Tournament Journey**: A chronological timeline tracking Group Stages, Survival Stages, and Grand Finals.
- **Prize Pool & Awards**: Detailed breakdowns of prize distributions and special awards (FMVP, Gunslinger, etc.).
- **Venue & Hosting**: Historically accurate hosting details with integrated Google Maps directions for venues like Ülker Sports Arena (Istanbul) and ExCeL London.

## 🛠️ Technology Stack
- **Frontend**: HTML5, Vanilla JavaScript (ES6+), CSS3 (Custom Design System).
- **Data Architecture**: Structured JSON datasets for each tournament year.
- **Animations**: Glitch-text effects, floating logo animations, and a dynamic canvas-based particle network.
- **Asset Management**: Hybrid local and proxy-based image delivery for high-performance logo rendering.

## 📦 Getting Started
1. Clone the repository: `git clone <repo-url>`
2. Install dependencies: `npm install`
3. Run locally: Use a live server (e.g., VS Code Live Server) or simply open `index.html`.

## 📊 Data Source
All tournament data, team information, and logos are sourced and verified against **Liquipedia**.
